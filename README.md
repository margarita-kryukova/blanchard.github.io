# Blanchard Project

## Описание

Blanchard — это [веб-приложение](https://margarita-kryukova.github.io/blanchard.github.io/), разработанное в рамках финальной работы курса Веб-вёрстка «Базовый уровень». Приложение предоставляет пользователю возможность взаимодействовать с данными и предоставляет функциональность для работы с состоянием приложения.

## Функции

- Ответственный и адаптивный дизайн
